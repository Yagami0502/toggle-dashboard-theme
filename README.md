# toggle-dashboard-theme

A Codex skill for adding or refining a persistent light and dark theme toggle in a dashboard UI.

## What it captures
- Single-source theme state
- Root `dark` class management
- Local storage persistence
- Fixed-corner control placement

## Install
1. Clone or download this repository.
2. Copy the repository folder into `~/.codex/skills/toggle-dashboard-theme`.
3. Start a new Codex session.
